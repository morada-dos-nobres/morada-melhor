Raio-X da Opinião do Morador (Guia de Elaboração)

Este documento serve como base técnica para a criação dos formulários de consulta aos moradores. O objetivo é mapear a realidade do residencial em profundidade, gerando a Matriz de Consenso para o novo Estatuto e Regimento Interno.

Estrutura da Consulta

Para cada área temática, a consulta deve iniciar com a pergunta padrão de satisfação:

“Em uma escala de 0 a 5, qual o seu nível de satisfação com os serviços prestados pela gestão atual nesta área?” (0 = Muito Insatisfeito | 5 = Muito Satisfeito)

As pesquisas detalhadas foram movidas para o diretório pesquisas/ para facilitar a gestão e expansão de cada tema.

Áreas Temáticas (Links para Pesquisas)

  1. 0. Reafirmação de Princípios e Legitimidade
  2. 1. Administração e Transparência
  3. 2. Distribuição de Água
  4. 3. Segurança
  5. 4. Manutenção e Serviços Operacionais
  6. 5. Convivência e Regimento Interno

Próximos Passos

Para cada área, definiremos as perguntas de múltipla escolha e os espaços para sugestões que comporão o formulário final.
